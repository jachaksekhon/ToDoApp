A basic to do application that runs on your local machine

Requirements:

CD into the api folder and: 
{
	Download and install Express JS : 'npm i express' in command line
	Download and install Mongoose : 'npm i mongoose' in command line
	Download and install cors : 'npm i cors' in command line
}

Download and install MongoDB: https://www.mongodb.com/try/download/community 

Download and install MongoDB Shell: https://www.mongodb.com/try/download/shell

Open up the MongoDB Shell and run 'mongo'
The output should provide you with a 'connecting to: mongodb://xxx.x.x.x:xxxx'
Copy and save this to put under the 'connectionString' variable inside of server.js in the api folder

Download and install nodemon : 'npm i -D nodemon' in command line

Running:

npm start under "api" to start the connection to the DB and local server

npm start under  "client" to open up the local server (your todo list)

Frameworks and databases used:
- MongoDB
- Mongoose
- Express JS
- React JS
- Node JS
